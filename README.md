# FitTrack - Premium Fitness Tracker

A full-stack fitness tracking application with a premium dark-mode aesthetic.

## Folder Structure
- **Backend**: Node.js/Express API with MongoDB/Mongoose.
- **Frontend/Fitness_Tracker**: React (Vite) application.

## Getting Started

### 1. Backend Setup
1. Open a terminal in the `Backend` directory.
2. Run `npm install` to install dependencies.
3. Ensure MongoDB is running on your machine (default: `mongodb://localhost:27017/fittrack`).
4. Run `npm run dev` to start the server on [http://localhost:5000](http://localhost:5000).

### 2. Frontend Setup
1. Open a terminal in the `Frontend/Fitness_Tracker` directory.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the React app on [http://localhost:5173](http://localhost:5173).

## Features
- **Dashboard**: View daily stats and recent workouts.
- **Workouts**: Log new workouts and view history.
- **Progress**: Track weight and performance metrics.
- **Nutrition**: Monitor macros and hydration.
- **Profile**: Manage your fitness profile.
